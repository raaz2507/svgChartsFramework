# 📊 SVG Chart Framework

A lightweight, flexible, and dependency-free charting library built using pure JavaScript and SVG.

> Designed for developers who want full control over chart rendering, animations, and interactivity.

---

### Testing Link 
https://raaz2507.github.io/svgChartsFramework/svgChartFrameworkExample.html

- 📸 Screenshot (chart image)

- 🎥 Demo GIF  

## ✨ Features

- 📈 Line & Area Chart Support
- 🎯 Interactive Data Points
- 💬 Custom Tooltip System
- ⚡ Smooth Animations (SVG + CSS)
- 📅 Dynamic Labels (24 Hours / Weekly)
- 🧩 Modular Architecture (ES Modules)
- 🚫 No External Libraries
